# Statistical Analysis of Social Attitudes

## Overview
This project analyzes a large social survey dataset to examine relationships between demographic characteristics and public opinion using statistical methods.

## Tools
- Statistical Modeling
- Hypothesis Testing

## Dataset
- 75,000+ survey records

## Objectives
- Analyze demographic trends
- Evaluate relationships between income, education, and public opinion
- Apply statistical inference techniques

## Results
- Identified meaningful relationships between demographic variables and survey responses.
- Collaborated with a team of three to complete the analysis and present findings.
